## 📌 Project Overview
This repository contains the complete design, mechanical analysis, and functional prototyping of an **Overhead Valve (OHV) pushrod mechanism** for a 328 cc naturally aspirated engine. This project was developed as part of the Internal Combustion Engines course at the Arab Academy for Science, Technology and Maritime Transport (AASTMT).

### Engine Specifications:
- **Capacity:** 328 cc
- **Compression Ratio:** 9:1
- **Estimated Power:** 7.0 kW (Brake Power) @ 80% efficiency.
- **Prototype Materials:** PLA/ABS (3D Printed).
- **Industrial Materials:** Forged Steel.

## 📁 Repository Structure
To ensure easy navigation, the project is organized into the following directories:

* **`Report/`**: Contains the final technical report in PDF format and the LaTeX source code.
* **`CAD-Models/`**: Includes the 3D designs created in **SOLIDWORKS** (`.SLDPRT`, `.SLDASM`) and universal `.STEP` files.
* **`MATLAB-Simulation/`**: Contains the **Simulink** models (`.slx`) and MATLAB scripts (`.m`) used for kinematic and dynamic engine analysis.

## 🛠 Tools & Technologies
- **CAD Modeling:** SOLIDWORKS (Mechanical design and assembly).
- **Simulation:** MATLAB & Simulink (Valve timing and combustion dynamics).
- **Documentation:** LaTeX (Scientific reporting).
- **Manufacturing:** FDM 3D Printing (Prototyping).

## 🚀 Key Results
- **Structural Integrity:** Validated safety factors for the camshaft, pushrod, and crankshaft using both analytical calculations and simulation.
- **Kinematic Validation:** Successful synchronization of valve timing through Simulink models.
- **Cost-Effective Prototyping:** Demonstrated that a functional educational model can be produced for approximately 700 EGP using additive manufacturing.
